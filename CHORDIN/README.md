# CHORDIN
CHORDIN: MIDI信号からコードを検出するVSTプラグイン。

## 主な機能
- MIDI信号からのコード検出
- MIDI信号からのノート検出
- MIDI信号からの度数検出
- 転回形の対応
- スケール構成音のメモ機能

## 非公開部分:
- プラグインのEditor部分（UI）
